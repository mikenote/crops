# 精准扶贫视域下农村电商资源对接平台设计-以“果农无忧” 微信小程序商城为例
摘要：随着"互联网+"时代的到来,依托电商平台促进农村资源对接成为了电商扶贫的重要助力。本文在阐述农村电商发展问题与现状的基础上,提出了构建"果农无忧"微信小程序,并分析了构建平台的建设技术,帮助果农推广滞销水果,提供农产品鲜果在市场上的供求信息,以期提高农业生产效率,为果农增收创收,实现精准扶贫。 关键词: 精准扶贫；电商平台；农产品滞销

# 二、系统介绍
 1. 我国人口基数大,改革开放以来我国虽然减少了2.1亿贫困人口,但是我国的反贫困事业还远远未完成,据国务院扶贫办统计,截至2014年,我国仍有12.8万个贫困村,7017万的贫困人口,这些贫困人口主要分布于缺乏生存条件的中西部山区。经调查发现,这些山区普遍存在自己的农业产业,其中,中部地区的果业尤为突出,但因其交通闭塞,互联网信息落后,农民虽拥有大量的生产优势,却无法将自己的产品推入市场。
 2. 电商扶贫是指引导和鼓励第三方电商企业建立电商服务平台,注重农产品上行,促进商品流通,不断提升贫困人口利用电商创业、就业能力,拓宽贫困地区特色优质农副产品销售渠道和贫困人口增收脱贫渠道,让互联网发展成果惠及更多的贫困地区和贫困人口。将互联网时代日益主流化的电子商务纳入扶贫开发工作体系,创新了扶贫开发方式,有利于促进农产品信息的交互,为提升精准扶贫的成效提供了极大的助力。然而农村电商在实践中仍面临了诸多问题:由于地域、经济发展等原因,农村的生活水平相对城镇较为落后,农民的思想观念较为保守,在接受新鲜事物方面存在一定的难度;信息基础设施是网上交流的前提,是开展电子商务的先决条件,农村网络体系不健全,网络知识缺乏,信息基础设施较差;农村物流体系不完善,物流配送欠佳,基于我国地域广阔和人口分布的情况,部分地方交通落后,物流网络不发达,造成了农村电商在物流配送方面的难题;产品品牌不响亮,同质化现象明显,品种单一、模式僵硬、产品同质化等问题。
 3. “果农无忧”电商扶贫项目以扶贫助困公益行动为载体,运用电子商务专业知识构建“果农无忧”电商平台,利用微信公众号等新媒体进行公益推广,“果农无忧”电商扶贫项目以扶贫助困公益行动为载体,运用电子商务专业知识构建“果农无忧”电商平台,利用微信公众号等新媒体进行公益推广,致力于为果农与水果经销商和有需求的消费者服务,为其提供专业的滞销产品销售买卖平台,立志解决果农滞销问题。通过整合果农滞销信息的整理、发布,形成果农与水果经销商的连接枢纽。果农滞销信息的收集、推广与发布、让果农与经销商可以有直接的联系;通过精深加工,使原始果品加工成高端产品,提高水果的附加值;在平台发布助农惠农政策,提供相关的政策解读。
## 需求分析
### 1.用户分析
- “果农无忧”平台的用户分为供应商，需要解决滞销果品售卖问题的果农;滞销果品的买家，对干果品有需求的个人消费者或企业。“果农无忧”平台主要针对滞销果农，利用“果农无忧”平台帮助水果滞销果农发布信息与推广以解决水果滞销问题，滞销水果由政府背书(政府公信力确保质量)后，发布至“果农无忧”电商平台,由“果农无忧”项目运营推广至个人消费者或企业:对干水果消费者,目前只针对无锡辣塘职教园在校的所有师牛。平台管理者在“果农无忧”平台将滞销水果信息提供给水果消费者，消费者可以以低干市场价的价格买到优质的水果,同时为公益献出了一份力量。
### 2.平台构建必要性
- 农产品的深销会带来一定的社会危害性 而目前除了政府的农产品滞销整扶官传平台 或通讨相关网络软文来寻求大型由商和消费者的帮助 尚未有直正成熟的推广滞销农产品电商平台，构建“果农无忧”电商平台为果农提供正确的信息来源及时了解农产品鲜果在市场上的
供求信息,价格情况通过移动互联打开滞销果品的销路,努力排除农产品滞销的隐患,做到真正电商扶贫。
### 3.类似平台对比
- 经互联网大数据查询 米似平台有“拼多多的“拼”模式助农扶贫·微信公众县“知农网”的滞销农产品的软文推送，据调查 拼多多是用了年的时间，依赖大型电商平台，为分散的农产品整合出一条快速通道，称为拼多多“绿色通道”确保了农产品的产销对接“知农网”通讨微信的软文推送,为订阅了此公众号的人群推送滞销农产品的相关信息，为此呼吁广大人群帮助解决。但是现有的大多数农业公益信息交流平台主要是以信息发布为主要的模式，并非严格意义上的公益脱贫项目。本项目依托微信小程序平台，将解决果农滞销问题作为项目主旨，运用互联网进行宣传和运作.农产品通过原产地直供的销售模式，实现农产品差异化竞争。原产地直供的销售模式的优势在于:一方面.营销模式灵活采用了“其地+传统B2C+故事性营销”方式 不再依赖大型电商平台 且有较强的自主性，灵活件·另一方面产品保质新鲜程度高消费者下单后,能从原产地发货,保证买家能在最短时间内收到商品,助于提高消费者的满意程度与购物体验。


## 平台设计思路
“果农无忧”平台以扶贫助困行动作为载体，运用电子商务专业知识，基于微信小程序的基础上，为水果滞销的农户解决困难所建立的“果农无忧"平台。平台的设计分为前台功能模块设计和后台功能模块设计2个部分。
### 1.前台功能模块设计
前台模块的设计分为4个一级子模块和3个二级子模块,前台功能模块如图所示
![在这里插入图片描述](https://img-blog.csdnimg.cn/65946d826f5e45b096cac0099699398b.png)
- **爱心助农**:在该模块中主要展示的是国家对于滞销农产品和电商扶贫的一系列相关政策，并目提供相应的政策解读。其中还分为3个一级子模块:扶贫故事，爱心农户、项目介绍。
- **扶贫故事**:发布各地果农的滞销农产品的相关问题及解决情况，管理员将相关信息发布在
平台上，帮助各个地区的果农解决滞销果品的销售问题。
- **爱心农户**:对各地的困难农户进行的走访调查与爱心帮助以软文推送的方式进
行宣传。
- **项目介绍**:对“果农无忧”扶贫助农项目的简要介绍。
- **爱心水果**:主要展示的是在当季所滞销的优质水果产品信息。滞销水果由政府背书(政府公信力确保质量)后,将详细信息发布至“果农无忧”微信小程序平台以便干个人消费者与企业进行购买。
- **千果零食**:为深销水果提供更多的出售方式，把滞销果品制作成果干果脯类零食等，为消费者提供更多的购买类别。
- **明星组合**:展示当季水果之间相互搭配健康营养的食用方法,为消费者提供更多选择,以带动滞销水果产品多种类别的搭配销售。
### 2.后台功能模块设计

后台功能模块的设计分为6个子模块.后台功能模块如图：
![在这里插入图片描述](https://img-blog.csdnimg.cn/9362547ec2c645b5b063c40ee31d3fbc.png)
- **滞销水果产品信息的管理**：发布滞销水果产品信息、修改滞销水果产品信息和删除滞销水果产品信息。
- **滞销水果产品分类管理**：对滞销水果产品分类进行维护,包括了添加分类、修改分类和删除分类。
- **助农扶贫政策管理**：发布、查询、修改和册除助农扶贫的政策信息，并对于相关政策做出解读。
- **爱心助农信息管理**：对爱心助农的信息进
行维护。对于给予帮助的爱心人士或企业进行管理。
- **售出商品订单与支付管理**：对消费者提交的订单进行管理，处理和支付控制功能
- **物流信息管理**：售出的产品订单反馈到滞销水果的所在地物流公司，管理员对物流快递人员和物流快递信息进行维护农产品的滞销会带来一定的社会危害性,而目前尚未有真正成熟的推广滞销农产品电商平台，同时滞销水果虚假信息难以分辨，需要构建发布滞销水果真实信息的销售平台。“果农无忧”平台以扶贫助困行动作为载体，运用电子商务专业知识,其干微信小程序，为水果滞销的农户解决困难。构建“果农无忧”电商平台.为果农提供正确的信息来源.及时了解农产品鲜果在市场上的供求信息与价格情况.排除农产品滞销的隐患,同时通过移动互联打开滞销果品的销路,促进农村电商资源对接,直正实现精准扶贫。
